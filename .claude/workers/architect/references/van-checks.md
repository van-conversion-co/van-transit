# Van Model Spatial Checks

Run before layout planning when Van.md has make, model, year, and wheelbase populated.

---

- **Slider door position**: confirm which side (driver's side or passenger's side). Affects kitchen zone placement — tall units must not obstruct the door opening arc.
- **Rear door type**: barn doors vs tailgate. Barn doors require a full clearance arc behind the van — any rear-mounted furniture or bed that extends to the doors must account for the door swing, not just the opening width.
- **Wheel arch profile**: confirm arch height and width from Van.md measurements. Wheel arches on Transit, Sprinter, and Crafter differ significantly and determine whether under-arch storage is viable or lost space.
- **High roof vs standard roof**: standing height at centre vs sides differs. Note where headroom drops below standing height on each side — this constrains kitchen worktop height and locker depth on the sides.
- **Internal rib spacing**: van ribs are the primary fixing points for battens and furniture. Note approximate rib spacing in Van.md if not already recorded — Carpenter will need this.

Flag any of these fields as `-tbd-` in Van.md if not confirmed. Do not proceed with layout planning until door type, wheel arch dimensions, and roof type are confirmed.
