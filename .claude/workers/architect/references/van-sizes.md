# Van Size Reference

Use when selecting a van or sanity-checking Van.md dimensions against known model ranges.

---

**Van selection questions** — ask before suggesting a size if the owner has not yet purchased:

1. How many people will be living or staying in the van?
2. What type of trips — weekends, extended travel, full-time?
3. What equipment needs to be stored (bikes, surfboards, climbing gear, tools)?
4. What facilities are required — sleeping only, cooking, toilet/shower?

**Body shape note**: Ducato/Jumper/Boxer/Promaster have boxier, straighter sides — more usable width but shorter internal length. Sprinter/Crafter/TGE taper more at the top, trading some upper volume for a longer floor (LWB 4.3m, XLWB 4.7m). For a bed across the van, boxier gives more width; for a lengthways bed, Sprinter/Crafter offer more length.

When an owner provides corrected or physically measured dimensions that differ from these figures, update Van.md with the measured value and source. Over time, the Architect accumulates more precise community-verified data for specific make/model/year combinations.

---

**Large van weight note**: large vans have significantly heavier kerb weights than medium vans, leaving less headroom before the 3.5T GVW limit. A Sprinter LWB can kerb at ~2,100kg, leaving ~1,400kg for payload — which disappears quickly with a full conversion, occupants, water, and gear. If the owner is converting a large van, start weight tracking in Weight.md from day one and flag this tighter margin explicitly.
