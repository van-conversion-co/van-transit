# Cable Sizing

Use when sizing cables for any circuit. Always calculate — never estimate on cable size. Undersized cable is a fire risk.

---

## Sizing process

1. Identify: circuit current (amps), cable run length (metres, **round trip** — positive and negative wire both count), acceptable voltage drop (<3% for lights/electronics, <5% for motors).
2. Calculate: `cable cross-section (mm²) = (current × cable length) ÷ (voltage drop allowance × voltage)`
3. Always round up to the next standard cable size. Add a 25% buffer when in doubt — oversizing cable is cheap; undersizing is a fire risk.
4. Flag any run >5m as needing explicit calculation — don't estimate.
5. Recommend tinned marine-grade cable for damp environments (van floor, near water tank).

---

## Wire colour conventions (UK)

- **12V DC**: red = positive, black = negative
- **230V AC mains**: brown = live, blue = neutral, green/yellow = earth

Automotive and marine wiring historically uses red/black for 12V — acceptable in a van build provided it is consistent throughout. Never mix colour conventions on the same circuit.

---

## Practical tips to minimise voltage drop

- Keep wire runs as short as possible — route carefully, don't take the scenic route
- Ensure all connections are tight but not overtightened (can damage crimps)
- Check contacts are clean and free of corrosion
- Use proper crimped connections, not twisted joins — crimping is the standard for van wiring; soldering is more reliable long-term but harder to do well
- Use bus bars to collect multiple circuits rather than daisy-chaining connections at the battery terminals
